

    - 1️⃣ What is the difference between var, let, and const?
    ans:  var allows us hoisting but it give undefined if tried to access before assigning and we can reassign its value and also can redeclare it. let creats a temporal dead zone, if tried before assigning will throw an error and we can reassign its value but can't redeclare it. const also creats tempora dead zone and we can't reassing its value after assigned once.

    - 2️⃣ What is the spread operator (...)?
    ans:  spread operator can expand the elements of  array, object or string into individual properties. 

    - 3️⃣ What is the difference between map(), filter(), and forEach()?
    ans:  map() returns a new array and doesn't modify original array, filter() returns a new array and also doesn't modify original array, forEach() doesn't return new array and can modify original array if wanted.

    - 4️⃣ What is an arrow function?
    ans:  arrow function is a type of function which automatically  return a value if in case of single statement but return is used when multiple statement.
